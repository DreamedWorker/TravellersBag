快速开始教程

你将可以在阅读并在遵循本文的指引下完成旅者行囊的下载和安装。

系统要求

要求内容
系统版本最低要求 macOS 14.0（这是 GPTK 能够运行的基本要求）；建议使用最新的正式版系统。
设备架构x64 和 arm64 均可
可选内容Apple SF Fonts

使用注意事项：

  1. 应用严格遵循 Apple 的应用沙箱，不会读写未经你授权的容器外的任何文件
  2. 建议你始终使用稳定版系统体验应用，我们不能确保测试版系统是否有自身问题导致应用运行不如预期

下载应用

第一次使用

使用我们在 Github 上发布的 ZIP 文件，你可以直接解压运行。单击前往下载。

当然了，你也可以从我们的源代码中构建运行，这将不会出现「未知开发者」的「错误」弹窗，但这需要你具有一定的开发知识，存在一定的门槛哦。下面是使用源代码构建的一些建议事项，供你参考：

项目内容
分支main
Xcode 版本16.0
主程序 Swift 版本Swift 6*
小组件 Swift 版本Swift 5
macOS 版本macOS 15.2

请始终从 main 分支构建，因为它也是我们发布 release 时依赖的分支，它的代码及功能是完整的、经过初步测试的，其它分支会不定时有内容被合并。

重要

由于在开发时使用了 Swift 6 ，因此你必须使用 Xcode 16.0 或更新版本，这也意味着你需要至少搭载 macOS 15.0 或更新版本的 Mac 才能编译本项目。

更新应用

一般地，你始终都可以以「第一次使用」的方式更新旅者行囊，因为这种方式只影响 App 文件而不影响 App 的沙盒。但是，我们更加推荐你采用应用内更新的方式进行。你可以在屏幕顶部的菜单：「旅者行囊」--「检查更新」，并按照提示完成下载更新。

  • 只要你不手动操作~/Library/Containers和~/Library/Group Containers中的与旅者行囊相关的文件，你的个人数据将不受任何影响。

如果你是通过源代码编译安装的话，在我们发布新版本后，请先拉取我们在main分支的最后一次提交，然后再编译运行即可。

添加米游社账号

提示

对于第一次使用旅者行囊的旅行者，首次冷启动将会进入初始化引导页面，请先按照指引对应用做出基本设置，然后就能进入应用首页了。（这个过程可能会要求重启应用若干次）

在进入主页面后，在应用左边栏中选择第一个导航项账号管理进入社区账号管理页面。之后请点按应用顶栏右上角的按钮添加账号，此时会向你弹窗，有扫码登录和Cookie登录两种方式。

无论你使用何种方式登录，请先确定你已经使用米游社登录并使用过其设置过用户名并绑定了游戏账号再进行登录操作哦，否则旅者行囊将无法获取你的信息。

需要注意的是，你只能登录中国大陆的账号，对于海外账号（Hoyoverse），旅者行囊并不支持。

扫码登录
  1. 进入「米游社」，点按底部导航栏上的「我的」，再点按屏幕左上角的扫描按钮；
  2. 点按手机屏幕上的确认登录按钮；
  3. 点按Mac屏幕弹窗上的完成扫描，等待我们获取相关信息后会自动关闭弹窗并将你的账号添加到列表中。
Cookie 登录
  1. 你需要首先提取到你的账号 Cookie，它应当包含 stuid 、stoken 和 mid 三个参数，然后将其复制到弹窗中的输入框中
  2. 点按屏幕上的确认按钮，等待我们获取相关信息后会自动关闭弹窗并将你的账号添加到列表中。

注意

请妥善处分你的 Cookie，它的泄露可能导致你的账号受到攻击。

还需要提醒一点，如果你添加的这个账号是本应用的第一个账号，我们会直接将其设置为你的默认账号，在获取祈愿记录、我的角色时将只获取该账号中的信息。如果你还添加了其它账号，则需要通过切换默认账号的方式来切换现实内容。

相关信息

如果你使用的是最新版旅者行囊，即：v0.0.3-build:20241024，则没有这个问题。

完成

到此，你已经完成了进入应用主业务逻辑后的全部前置设置，现在可以畅行于macOS上的提瓦特大陆啦！

安装时可能遇到的问题

来源不可信

这个问题是通过 ZIP 直接解压运行导致的。

由于 Apple 对应用公证导致的，由于个人开发者账号未选择加入Enroll计划，故无法上传公证代码。你需要前往系统设置中选择仍要打开，在进行验证之后就可以继续使用了。

虽然本项目的源代码是公开可查的，但如果你因为这个弹窗导致对应用的安全性产生质疑，立即删除是你当下及时止损的最佳做法，因为此时应用还没有开始运行。

应用没有图标或图标不全

这个问题是由于 Apple 的 SF 字体未安装或未完全安装导致的。在一般情况下，你的系统是自带有一整套可用的图标字体的，如果你遇到了这个情况，请前往 Apple 的开发者网站下载相关的字体，并在重新登录后重新打开旅者行囊。
